# Development

## Frontend (\*)

1. Navigate to 'frontend' folder
2. Run 'npm install' in the terminal
3. Run 'npm run dev' to start server

## Backend (\*)

1. Navigate to 'backend' folder
2. Setup virtual environment
3. pip install dependencies from requirements.txt
